## [0.1.0] - 2024-01-01

### Added
- Initial release of Monokai Nebula dark theme
- Initial release of Monokai Nebula Light theme
- Comprehensive UI color coverage for all VS Code workbench elements
- Semantic highlighting support
- Terminal ANSI color palette
- Bracket pair colorization
- Git decoration colors
- Language-specific token color rules (Python, Rust, JavaScript/TypeScript, Java, C/C++, and more)
